# TWP
Это игра-головоломка для Android (также доступна исполняемая версия для Windows), которую я решил создать после прохождения игры Джонатана Блоу «The Witness». Я действительно восхищаюсь этой игрой и хотел большего. Поэтому я создал «The Witness Puzzles».

В оригинальной игре «The Witness» есть остров, который нужно исследовать, заполненный панелями, которые нужно решить. Панель (обычно) содержит сетку с точкой входа, точкой выхода и набором правил, и цель состоит в том, чтобы провести линию от точки входа до точки выхода таким образом, чтобы удовлетворялись все правила. Это не так просто, как кажется.

В моей игре я создал процедурный генератор таких панелей, поэтому я могу играть в эти головоломки бесконечно.

Технологии и прочее
.NET Framework 4.7 и C# 7.0
MonoGame Framework 3.6
Немного модифицированный BloomFilter для Monogame и шейдер XNA от Kosmonaut3d
Для компиляции проекта Android вам потребуется разработка мобильных приложений с использованием .NET (а именно Xamarin), а также компонент Android NDK (R13B) в Visual Studio.
Для компиляции ресурсов MonoGame необходимо установить шрифт Roboto.
Целевая версия Android SDK: API уровень 25.
Структура решения
TWP Android – проект сборки для Android, содержит только основную активность и конфигурации.
TWP Desktop – проект сборки для Windows, содержит только метод Main() и конфигурации.
TWP Shared – это первый из двух основных проектов, содержащий весь игровой код. Его код используется совместно проектами Android и Desktop.
TWPBaseLib – это второй из двух основных проектов, содержащий всю основную логику и алгоритмы.

TWPVisualizer – это скорее устаревший проект, где я тестировал функциональность BaseLib. Использовать его не следует. Графический интерфейс пользователя совершенно не является надежным.
